Siigo Hackathon

Siigo Hackathon 2020

At the Siigo hackathon of 2020, we had to create an API with my team partner Jimmer Hernandez to manage tables on a database with a basic CRUD. We used MySQL for the database and the micro-framework Flask to create the API. The idea behind the API was to simulate transactions between companies, costumers, and products.

Heippi Hospital

API for the Heippi hospital

This project is a backend development to provide support to a fictitious hospital. The API allow the login and creation of patients, doctors, the doctors can create medical observations and access to the medical observations of any patient usig their ID. To make this project I used Flask to provide the end points and firebase for the database.

  • Date: March 2020
  • Category: Backend
